コーディング

  • 「プログラミング」とほぼ同義
  • コンピューターの動きを書いていくこと
  • コンピューターが理解できる言葉（プログラミング言語）を使う

プログラミング言語

  • 人間の言葉（日本語や英語）と機械の言葉（0と1の世界）の中間の言葉
  • いろいろな種類がある（Wikipediaの「プログラミング言語の一覧」には300近くの言語が載っている）
    • C/C++
    • Java
    • C#
    • JavaScript
    • Ruby
    • Python
    • Swift
  • 万能なものはないので用途や特徴や好みで選んで使う
  • プログラミング言語っぽいけど厳密には違うものもある
    • HTML
    • CSS

コードを書く道具

  • キーボード
    • お気に入りのキーボードを持ち込む人も多い
    • キーの並び : JIS配列/US配列
    • 打鍵感: 静電容量無接点/メカニカル/パンタグラフ/メンブレン
    • 疲れない: 左右分割/エルゴノミクス
  • エディタ
    • 人によっていろいろな好みがある（時に宗教戦争が起きることも）
    • シンプルなエディタ: メモ帳(Windows), テキストエディット(macOS)
    • 歴史ある人気エディタ: Emacs, Vim
    • カスタマイズ自在なエディタ: Atom, Sublime Text...
    • 高機能なエディタ(IDE): InteliJ, Xcode, Visual Studio...

HTML

  • Webページを作るための言語
  • HyperText Markup Language
    • < >〜</ >の タグ でしるしをつける言語
  • いろんな種類のタグがある
    h1,h2,...,h7
    見出し・小見出しをつくる
    p
    段落 (paragraph) をつくる
    a
    他のページやページ内の部分にリンク (anchor) を張る
    strong
    強調する
    div, span
    それ自体では意味を持たないタグ

ちょっと派手にしてみる

<html lang="ja">
  <head>
    <title>はじめてのWebページ</title>
    <style>
      .title {
        font-size: 24px;
        font-weight: bold;
        color: #333333;
      }
      .greetings {
        font-size: 18px;
      }
      .name {
        color: #FF6666;
        font-weight: bold;
      }
    </style>
  </head>
  <body>
    <h1 class="title">ごあいさつ</h1>
    <p class="greetings">こんにちは、<span class="name">世界</span>！</p>
  </body>
</html>

  • HTMLでは<script>タグの中に書く
  • JavaScriptとJavaは名前が似ている別のもの
    • Javaも広く使われているプログラミング言語
    • 調べたり質問したりする時に注意！
  • がんばればゲームも作れます
    • スマートフォン向けゲームのなかにはJavaScriptで書かれたものも
    • 私も昔、ゲームを作ってニュースサイトに紹介されたことがあります

コードの解説

  • var yourName = window.prompt('あなたのお名前は？');
    • ユーザーに『あなたのお名前は？』の質問をして、答えをもらう
    • yourNameという変数を作って、その答えを代入
  • var nameElement = document.querySelector('.name')
    • HTMLの中から name というクラスのある要素（部分）を見つけてくる
    • nameElementという変数を作って、見つけた要素を代入
  • nameElement.innerText = yourName;
    • nameElementのなかの文字に、yourNameを代入

おみくじを作ろう(1)

<script>
  var omikujiButton = document.querySelector('.omikuji-button');
  var resultElement = document.querySelector('.result');
  omikujiButton.addEventListener('click', function(){
    // おみくじボタンをクリックするとここが実行される
    var score = Math.floor(Math.random() * 10); // scoreに0から9の整数が代入される
    var result = '';
    if (score === 9) {
      result = '大吉です';
    } else if (score > 2) {
      result = '吉です';
    } else {
      result = '凶です';
    }
    resultElement.innerText = result;
  });
</script>
